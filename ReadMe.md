# Next-JS-PostgreSQL Dashboard

A fast, scalable user management dashboard built with **Next.js** and **PostgreSQL**.  
Easily manage users and their status with real-time toggling of active/inactive states and the ability to delete users directly from the interface.

## Features
- User list display with ID, Name, Email, and Status.
- Toggle user status between Active and Inactive.
- Delete users with a single click.
- Responsive UI with light and dark mode options.
- Built with modern React hooks and server-side rendering using Next.js.
- PostgreSQL backend for reliable data storage.

## Tech Stack
- **Next.js** for the frontend and server-side logic.
- **PostgreSQL** as the database.
- API routes for handling user status and deletion.

## Usage
Clone the repo, set up your PostgreSQL database, and start the development server to manage your users efficiently.
